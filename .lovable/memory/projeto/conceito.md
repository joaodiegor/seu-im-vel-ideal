---
name: Conceito do Projeto
description: Plataforma de leilão reverso imobiliário em todo o Brasil.
type: feature
---
Plataforma de leilão reverso imobiliário focada no mercado brasileiro (escopo nacional), inspirada no modelo do reverselist.com. Compradores publicam pedidos descrevendo o imóvel desejado (tipo, estado, cidade, bairro, orçamento) e corretores cadastrados competem enviando propostas privadas. Funciona em qualquer cidade do Brasil — o formulário de busca usa hierarquia Estado → Cidade → Bairro com dados mockados das principais capitais e cidades. Anteriormente o projeto era restrito a São Luís (MA) sob o nome "SLZ Imóveis", mas foi expandido para o Brasil inteiro como "Brazuka Imóveis".
